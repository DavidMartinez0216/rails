== Integration Test App -- Integration Testing for Rails

Integration Test App is a Rails app inside Rails that tests core features
of Rails. It is meant to catch regressions that would not be found by the
gem unit tests. Its Gemfile points to the local Rails, and tests covered
by `rake test` will be included in the continuous integration suite.


== Download and installation

Source code can be downloaded as part of the Rails project on GitHub:

* https://github.com/rails/rails/tree/master/integration-test-app
